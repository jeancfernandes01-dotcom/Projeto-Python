# Projetos de Automação e Lógica - Python 

Repositório dedicado ao estudo de lógica de programação e automação de processos utilizando **Python**.

##  Projetos Desenvolvidos:
* **Barbearia.py:** Sistema para gestão de preços e serviços (focado em regras de negócio reais).
* **Cofrinho.py:** Calculadora de Reserva Financeira para auxiliar no planejamento pessoal.
* **LojaRoupaFeminina.py:** Automação de condições de pagamento e descontos para varejo.
* **AprovadoExameReprovado.py:** Lógica de decisão baseada em médias e assiduidade escolar.

##  Melhorias Recentes:
- Ajuste na lógica de cálculos combinados no sistema da Barbearia.
- Implementação de verificação de assiduidade nos sistemas de ensino.

##  Objetivo:
Desenvolver raciocínio lógico e aplicabilidade da programação em problemas do cotidiano.
